# 요구 사항

## 검색 결과 화면 (메인 화면)
- [ ] 검색어를 입력할 수 있습니다.
- [ ] 검색된 이미지 리스트가 나타납니다. (각 아이템에는 이미지와 함께 날짜와 시간을 표시)
- [ ] 스크롤을 통해 다음 페이지를 불러옵니다.
- [ ] 리스트에서 특정 이미지를 선택하여 '내 보관함'으로 저장할 수 있습니다.
- [ ] 이미 보관된 이미지는 특별한 표시를 보여줍니다. (좋아요/별표/하트 등)
- [ ] 보관된 이미지를 다시 선택하여 보관함에서 제거 가능합니다.
- [ ] 필터 토글 기능을 사용할 수 있습니다. (전체/이미지/영상)
- [ ] 다른 화면에서 돌아와도 스크롤 상태가 유지됩니다.

## 보관함 화면
- [ ] 검색 결과에서 보관했던 이미지들이 보관한 순서대로 보입니다.
- [ ] 보관한 이미지 리스트는 앱 재시작 후 다시 보여야 합니다.
